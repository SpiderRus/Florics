# 🌿 Демонстрация Markdown в GreenDecor

## Обзор

Для растения **"Монстера деликатесная"** (ID: 1) созданы улучшенные описания с использованием полного спектра Markdown-разметки.

## 📍 Где посмотреть

Откройте в браузере:
```
http://localhost:8080/goods/1
```

Перейдите на вкладки:
- **"Описание"** — детальное описание с форматированием
- **"Уход"** — подробная инструкция по уходу

---

## ✨ Что демонстрирует новая разметка

### В разделе "Описание" (`detailedDescription`):

#### 1. **Заголовки разных уровней**
```markdown
## О растении
### Основные характеристики
```
- H2 с зеленой линией снизу
- H3 с меньшим размером шрифта

#### 2. **Форматирование текста**
```markdown
**Монстера деликатесная** (*Monstera deliciosa*)
```
- Жирный текст для акцентов
- Курсив для латинских названий

#### 3. **Списки маркированные**
```markdown
- **Размер**: В природе до 20 метров
- **Тип роста**: Лиана с воздушными корнями
```
- Кастомный маркер 🌿
- Структурированная информация

#### 4. **Нумерованные списки**
```markdown
1. **Неприхотливость** — прощает ошибки
2. **Эффектный вид** — тропический оазис
```
- Автоматическая нумерация
- Форматирование элементов

#### 5. **Blockquote для важных заметок**
```markdown
> ⚠️ **Важно**: Листья содержат оксалат кальция
```
- Выделение критичной информации
- Визуальное отделение

---

### В разделе "Уход" (`careInstructions`):

#### 1. **Структура с эмодзи-заголовками**
```markdown
## 🌞 Освещение
## 💧 Полив
```
- Эмодзи сохраняются
- Логическое разделение секций

#### 2. **Подзаголовки и параграфы**
```markdown
### График полива:
**Весна-лето**: 1-2 раза в неделю
```
- Трехуровневая иерархия
- Жирный текст для ключевых слов

#### 3. **Таблица проблем**
```markdown
| Проблема | Причина | Решение |
|----------|---------|---------|
| Желтеют листья | Перелив | Отрегулируйте полив |
```
- Табличное представление данных
- Удобная структура

#### 4. **Горизонтальные разделители**
```markdown
---
```
- Визуальное разделение секций
- Улучшенная читаемость

#### 5. **Цитаты-советы**
```markdown
> 💚 **Следуя этим правилам, ваша монстера будет радовать вас...**
```
- Итоговое сообщение
- Мотивирующий финал

## 🎨 Использованные стили

### Цветовая схема проекта:
- **Заголовки H1**: `--forest-green` (#2d5016)
- **Заголовки H2**: `--sage-green` (#4a7c2c)
- **Ссылки**: `--sage-green` с hover на `--leaf-green`
- **Код**: фон `--light-green` (#e8f5e0)
- **Маркеры списков**: 🌿 (тематика растений)

### Типографика:
- **Параграфы**: line-height 1.7 для читаемости
- **Жирный текст**: font-weight 600
- **Курсив**: цвет `--sage-green`
- **Код**: Courier New, monospace

## 📊 Сравнение до/после

### ДО (простой текст):
```
Монстера деликатесная родом из тропических лесов...
```
- Одна сплошная стена текста
- Нет визуальной иерархии
- Сложно находить нужную информацию

### ПОСЛЕ (Markdown):
```
## О растении
**Монстера деликатесная** (*Monstera deliciosa*)

### Основные характеристики
- **Размер**: В природе до 20 метров
```
- Четкая структура
- Быстрая навигация
- Приятное чтение

## 🧪 Тестовые примеры

### Все поддерживаемые элементы:

1. **Заголовки**: # ## ###
2. **Жирный**: \*\*текст\*\*
3. **Курсив**: \*текст\*
4. **Списки**: - элемент
5. **Нумерация**: 1. элемент
6. **Ссылки**: [текст](url)
7. **Код**: \`inline\` и \`\`\`block\`\`\`
8. **Таблицы**: | col1 | col2 |
9. **Blockquote**: > цитата
10. **HR**: ---

### Обратная совместимость:
- ✅ Эмодзи сохраняются (🌞, 💧, 💨, 🌡️, 🌱, 🪴, ✂️, ⚠️, 💚)
- ✅ Переносы строк работают (\n\n)
- ✅ Существующие данные не ломаются

## 📝 Как редактировать

Откройте файл:
```
backend/src/main/kotlin/com/example/webflux/repository/GoodsRepository.kt
```

Найдите нужный товар (например, строка 18 для Монстеры):
```kotlin
detailedDescription = """
## Ваш заголовок
Ваш текст с **форматированием**
""".trimIndent()
```

После изменений:
1. Пересоберите backend: `mvn clean install`
2. Перезапустите приложение: `mvn spring-boot:run`
3. Обновите страницу в браузере
